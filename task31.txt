3.1. Что такое sql injection? Приведите пример атаки. Какие методы защиты от нее вы знаете?

SQL injection - это один из самых доступных способов взлома сайта. Взломщик передает на сайт запрос. В запросе содержится подстрока, которая выполнится в базе данных.

Пример. Допустим, клиент отправляет GET-запрос https://site.ru/get_news?id=123
В коде мы получаем id из запроса и отправляем запрос серверу БД.
"SELECT * FROM news WHERE id_news = $id"

Взломщик вместо "123" может передать строку
12;INSERT INTO admin (username, password) VALUES ('mothers_hacker_777', 'foo');

Если в коде нет обработки входных данных, то юный взломщик добавит нового пользователя в таблицу админов сайта.

Методы защиты:
валидация входных параметров
экранирование спецсимволов
усечение параметров
параметризированные запросы (в Perl - DBI::quote или DBI::prepare)
